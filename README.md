# Tetris-React
A Tetris game made using ReactJS and SASS. The game was deployed to GitHub Pages: https://hdn7.github.io/tetris-react/

# Instalation
```
git clone https://github.com/hdn7/tetris-react.git
cd tetris-react
npm install
```

# Available Scripts
#### npm start
#### npm build
#### npm predeploy
#### npm deploy
